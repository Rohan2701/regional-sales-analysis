# Regional Sales Analysis (New Zealand)

## Objective
To analyze B2B regional sales data to determine revenue trends, geographical performance, and sales channel distribution. 

## Tech Stack Used
* **MS Excel:** Data cleaning, formatting, and handling UTF-8 encoding issues.
* **MySQL:** Relational database engineering, Star Schema design, and bulk data loading using `LOAD DATA INFILE`.
* **Power BI:** Data modeling, DAX measure creation, and interactive dashboard visualization.

## Key Insights
1. **Global Revenue:** The dataset reflects $128.55M in total USD revenue across 67,579 orders.
2. **Channel Dominance:** The Wholesale channel makes up the majority of the order volume (53.76%).
3. **Data Anomaly Addressed:** The sharp decline in the Q4 2019 trendline is due to the dataset containing incomplete records for December 2019 (only 12 days of data), rather than a structural business failure.
